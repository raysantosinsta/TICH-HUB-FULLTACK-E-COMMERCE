# 📝 Breve Descrição da Loja

A loja online **ShopHub** foi desenvolvida com o objetivo de oferecer uma experiência moderna,
rápida e intuitiva para os usuários.

O sistema simula um e-commerce real com funcionalidades essenciais, incluindo:

- Navegação intuitiva
- Interface moderna
- Experiência premium
- Responsividade completa
- Animações suaves
